About Git and GitHub

Git is a Distributed Version Control System (DVCS) for managing source code and projects, developed by Linus Torvalds in 2005, while GitHub is a service that provides a public repository for developers can store and check for any changes of their codes, using Git.

Git and GitHub can help you manage your codes easier and more convenient since they keep all your codes versions if you accidentally delete or make any changes to your codes. Moreover, they also improve cooperation among developers while working on the same project and provides numerous other services which are really helpful.

Basic Git Commands

When you have finished installing three apps above, open Visual Studio Code, move your mouse to "View", find "Terminal" and click on to open it.

Now I will show you some basic Git commands and steps to push your website onto GitHub.

I. Config

1. git config --global user.name 'yourusername' 2. git config --global user.email 'youremail'

First you will have to type in these two commands, using the username and email you registered on GitHub --> Enter. You only have to do this once

II. Clone

git clone 'URL'

Next you will go to your GitHub page, click on the “Code” section, find the “Code” box and copy the URL below that. Then you go back to the terminal and type in the command above --> Enter

III. Add, Commit, Push

1. git add . 2. git commit -m "comment" 3. git push

To create a new file, move your mouse to “File” section then click on “Open folder”. Choose the folder that you have just cloned (yourusername.github.io) and click “Select folder”. You can start coding your website by clicking on “New file” in the folder you just open.

Once you finish coding your website, you will have to push it on GitHub. Type in three commands above by that order then you are done!

IV. Branch

1. To create a new branch, you type in the command below:

git branch 'name of your new branch'

2. If you want to switch to the new branch you have just created or any other branch, use the command below:

git checkout 'name of that branch'

3. After working on another branch, if you want to join that branch with the main branch, this is the command:

git merge 'name of that branch'
